File Format Guide
Required Columns:
  • email - User email address
  • first_name - User first name
  • last_name - User last name
Optional Columns:
  • username - Username (defaults to email prefix)
  • phone - Phone number
  • gender - male/female/other
  • date_of_birth - YYYY-MM-DD format
  • address - Full address
  • emergency_contact - Emergency phone
  • blood_group - Blood group
  • employee_id - Employee ID (for staff)
  • roll_number - Roll number (for students)
  • department - Department name
  • qualification - Educational qualification
  • specialization - Field of specialization
  • experience - Years of experience
  • aadhar_number - Aadhar number
  • pan_number - PAN number

Note: Missing optional fields will be automatically filled with 'na' or default values.
